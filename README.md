## Run Locally

* Clone the project and go to the project directory

```bash
  cd tiktok-data-scraper-rag-recommender
```

* Install dependencies using `uv` (or install dependencies manually using pip from `pyproject.toml` file)
```bash
  uv sync
```
* Setup LLM API keys and base url in `.env` file in root directory

* Run script

```bash
  python main.py
```
